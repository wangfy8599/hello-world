# Branch

## Delete local branch
```
git branch -d <branch>
```

## Delete remote branch
```
git push origin --delete <branch>
```

# Tag
```
# list tag
git tag -n

# show tag detail
git show <tag_name>

# push all tags
git push origin --tags

# push one tag
git push origin <tag_name>

# delete tag
git tag -d <tag_name>

# commit tag
git push --delete origin <tag_name>
```
